# Data License Status

OpenInterstate publishes OSM-derived public dataset releases.

## Current Status

Public dataset releases are derived from OpenStreetMap and published under the
Open Database License (ODbL 1.0). This is declared in `datapackage.json` and
in each release manifest's attribution field.

When consuming releases:

1. attribute OpenStreetMap; the data is OSM-derived and ODbL obligations apply
2. do not assume all repository contents are approved as public data releases
3. do not assume all derived datasets have been separated from non-OSM overlays

## Release Expectations

Every OSM-derived release carries:

1. OpenStreetMap attribution
2. clear lineage statements (`source_lineage.json`, manifest lineage fields)
3. explicit separation of any non-OSM overlays
4. release-level metadata that states the applicable obligations
