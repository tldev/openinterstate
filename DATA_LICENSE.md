# Data License Status

OpenInterstate intends to publish explicit data-license terms for every public
release.

## Current Status

The data-license posture is still being finalized.

The first public dataset release has been published, but the long-term public
data-license language still needs to be formalized at the release-policy level.

Until the release policy is finalized:

1. treat public dataset artifacts as OSM-derived releases that require attribution
2. do not assume all repository contents are approved as public data releases
3. do not assume all derived datasets have been separated from non-OSM overlays
4. treat `docs/licensing_and_attribution.md` as the current working note

## Working Direction

The project assumes that OSM-derived releases will need:

1. OpenStreetMap attribution
2. clear lineage statements
3. explicit separation of any non-OSM overlays
4. release-level metadata that states the applicable obligations
