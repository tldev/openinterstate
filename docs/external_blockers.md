# External Blockers

This file exists to keep the program honest about what can and cannot be
completed by repository work alone.

## Legal And Licensing

1. choose the final code license
2. confirm the release posture for OSM-derived datasets
3. finalize attribution text for public releases and API responses

## Deployment

1. connect `openinterstate.org` to the chosen hosting platform
2. configure DNS for the live site
3. choose and provision the public API runtime

## Data Production

1. export the first named public release from real Pike-derived source data
2. validate release contents against the documented contract
3. publish checksums and attribution with the real artifact set

## Community

1. enable the preferred public support surface
2. publish launch materials
3. collect external feedback from the first real users
