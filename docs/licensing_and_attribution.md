# Licensing And Attribution

OpenInterstate is in pre-launch licensing review.

## Current Position

No final code license or final data license is declared in this repository yet.
Until explicit repository-level license files are added, do not assume a final
public license choice has been made for code or data in this repo.

## Working Assumptions

1. OpenStreetMap-derived data requires attribution.
2. Derived database outputs may carry Open Database License obligations.
3. Any non-OSM overlays must be clearly separated from OSM-derived outputs.
4. Public releases must include attribution and lineage notes.

## Required Before First Public Data Release

1. choose a repository code license
2. document the data license posture
3. publish attribution language
4. identify which artifacts are OSM-derived
5. review trademark and branding usage for third-party place names and logos

## Practical Rule

If a release cannot explain where its data came from and what obligations apply,
it is not ready to publish.
